PSP MANAGER README

PSP Manager is gemaakt door Hilco, Huub, Marijn, 
Patrick en Wouter als school opdracht.

De applicatie maakt twee Register waaren aan in
het volgende sleutel:

	HKEY_LOCAL_MACHINE\Software\PSPManager

Dit zijn de waarden:

	Open
	Starts

Open wordt gebruikt om aan te geven dat de 
applicaite aan staat. Mocht de applicatie
door onbekende reden niet correct sluiten,
zet deze waarde op 0.

Starts bewaard het aantal start van het
programma.
